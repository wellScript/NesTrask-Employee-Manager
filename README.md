# EmployeeDatabse
My project is a simple employee manager software. Using a graphical user interfae,
the program will allow the user to create a new employee, modify an existing
employee, or terminate the employee. Additionally, the user can search for
employees by name or employee ID, as well as export a list of all employees.
## Features
- Create new employees
- Modify existing employees
- Disable employees
- Search for employees
- Export employees
## Enhancements
- Allow storing of employee information via CSV file
- Allow exporting of employee list
- GUI Development
